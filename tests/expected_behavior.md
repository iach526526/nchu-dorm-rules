# Expected Agent Behavior by Question Type

## Dorm-Specific Violation (e.g., smoking, alcohol, mahjong, pets, cooking)

1. Check if the user specified a dormitory in their question.
2. **If no dormitory specified:** Ask the user which dormitory they live in (男宿, 女宿, 興大二村, 南投宿舍) before consulting any specific rule document.
3. **If dormitory specified:** Go directly to that dormitory's regulation document.
4. Consult `references/faq_map.md` for keyword matching to identify the correct dormitory-specific document and article number.
5. Use `references/index.md` to locate the correct local raw file path.
6. Read the specific article in `references/rules/` (or `references/rules_structured/`).
7. When citing, include the dormitory-specific document title, version date, article number, and PDF URL.
8. For appliance-related violations, cross-reference with the prohibited appliances list specific to that dormitory.

## General Repair / Maintenance

1. Do **not** ask which dormitory — repair procedures apply uniformly to all dormitories.
2. Consult `references/faq_map.md` under "repair" topic.
3. Read `references/rules_structured/Application-for-Student-Dormitory-Maintenance-Regulation-1131017.md` (ID: dorm-maintenance).
4. Identify the correct article (typically Article 4 for clogged toilets, general online repair system).
5. State the self-unclog requirement first (where applicable), then the official reporting procedure.
6. Include the source document title, version date, and PDF URL in the citation.

## Network / Internet

1. Do **not** ask which dormitory — network regulations apply to all dormitories (with the exception that Second Village has a dedicated network administrator).
2. Consult `references/faq_map.md` under "network" topic.
3. Read the primary regulation `references/rules_structured/Guidelines-for-the-Network-Management-of-Student-Dormitories-1071214.md` (ID: dorm-network).
4. For application procedures, also read `references/rules_structured/Dormitory-Network-Application-Guide.md` (ID: network-guide).
5. For Second Village network issues, additionally check `references/contacts.md` for the network administrator contact.
6. Include the relevant article number in citations.

## Refrigerator

1. Do **not** ask which dormitory — refrigerator regulation applies uniformly.
2. Consult `references/faq_map.md` under "refrigerator" topic.
3. Read `references/rules_structured/Directives-Governing-the-Use-and-Management-of-Student-Dormitory-Refrigerators-1141211.md` (ID: dorm-refrigerator).
4. Provide the relevant usage rules, item labeling requirements, and penalty provisions.
5. Include source document title, version date, and PDF URL.

## Air Conditioning Card

1. Do **not** ask which dormitory — AC card regulation applies uniformly.
2. Consult `references/faq_map.md` under "air_conditioning_card" topic.
3. Read `references/rules_structured/Guidelines-for-the-Management-of-Air-Conditioning-Cards-in-Student-Dormitories-1131217.md` (ID: aircon-card).
4. Describe recharge, card replacement, and usage rules as applicable.
5. Include source document title, version date, and PDF URL.

## Bicycle / Motorcycle

1. Do **not** ask which dormitory initially — the general bicycle/motorcycle regulation applies to all dormitories.
2. Consult `references/faq_map.md` under "bicycle_motorcycle" topic.
3. Read `references/rules_structured/Directives-Governing-the-Management-of-Bicycles-and-Motorbikes-at-Student-Dormitories-1101203.md` (ID: bike-motorcycle).
4. Supplement with dormitory-specific rules if the user asks about a specific dormitory (e.g., male dorm Article 26 has detailed parking rules).
5. Parking permit requirements, fines, and violation procedures are covered in the general regulation.

## Dorm Organization

1. Do **not** ask which dormitory — organization documents are not dorm-specific.
2. Consult `references/faq_map.md` under "dorm_organization" topic.
3. Read relevant documents:
   - `service-committee`: Articles of Association of Student Dormitory Service Committee
   - `committee-selection`: Selection and Evaluation Measures
   - `resident-rep-committee`: Articles of Association for Resident Student Representative Committee
   - `resident-rep-selection`: Measures of Resident Student Representatives Selection
4. Describe the committee's role, responsibilities, and selection process.
5. Include the relevant source document and PDF URL.

## Dorm Application

1. Do **not** ask which dormitory — application procedures apply uniformly.
2. Consult `references/faq_map.md` under "dorm_application" topic.
3. Read `references/rules_structured/Directives-Governing-the-Application-for-and-Management-of-Student-Dormitories-1131129.md` (ID: dorm-application).
4. Supplement with `references/rules_structured/Regulations-for-Dormitory-Accommodation-1131129.md` (ID: dorm-accommodation) for general accommodation rules.
5. Include source document title, version date, and PDF URL.

## Missing / Ambiguous Rules

1. When a user asks about a topic not covered or unclearly covered in any document:
   - State explicitly: "The provided documents do not contain a clear rule for this question."
   - Do not invent or assume rules.
   - Do not extrapolate from similar rules.
2. Optionally suggest the user confirm with the Dormitory Counseling Section or the responsible contact.
3. Look up contact information in `references/contacts.md` if appropriate.

## Cross-Dormitory Comparison

1. When the user compares rules across dormitories (explicitly or by omission):
   - If no dorm specified and the answer differs by dorm, ask which dormitory.
   - If the user explicitly asks for comparison, read all dormitory-specific rules on the topic.
   - Present each dormitory's rule separately.
   - Cite each dormitory's specific document with its own PDF URL.
2. Do not merge or average rules across dormitories.

## Cumulative Penalty Points

1. If the question is about cumulative thresholds (e.g., "how many points until eviction"):
   - Ask which dormitory if not specified.
   - The general pattern across all dorms: 15 points → cannot reapply next year; 20 points → immediate eviction.
   - However specific rules may differ (e.g., internet suspension thresholds).
   - Read the specific dormitory's Article 1 or equivalent general provisions.

## Public Property / Damage

1. Do **not** ask which dormitory — public property rules apply uniformly.
2. Consult `references/faq_map.md` under "public_property" topic.
3. Read `references/rules_structured/Dormitory-Public-Property-Management-Regulation-1040413.md` (ID: public-property).
4. Include source document and PDF URL.
