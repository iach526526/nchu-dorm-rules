# 中興大學宿舍規章 Skill

[Toc]

這是一個 AI skill ，讓大型語言模型可以根據宿舍的實際狀況做出回應。我只是盡量讓 AI 產生更準確的答案，AI 回覆可能有誤，這個專案的開發者不會對任何 AI 回覆的錯誤答案付任何責任，請自行判斷什麼時候應該找真人協助。這**只是讓你更快速掌握學校規定的工具**

## 功能說明
### 這個專案能幹嘛？
讓你習慣使用的大語言模型服務**正確的**告訴你正確的宿舍規定，你可以在不看任何規章的情況下丟一個問題，讓語言模型告訴你答案和應該確認的文件位置。

- 問到宿舍的規定
- 找到對應事務的負責人
  - ~~目前只能找到網管就是~~

### 這個專案不能幹嘛？
- X 讓你的室友有良好的衛生習慣
- X 

## 專案原理
就... AI skill ，通常我們叫這個方法 in-context learning 。就是我把規定的上下文告訴你的 AI ，讓 AI 讀完文件再回答。不過這個過程沒有任何 AI 受到「訓練」，如果你沒裝這個 skill ，那你的 AI 就會開始瞎講。如果你對機制很感興趣，可以從李宏毅老師的[【生成式AI導論 2024】第3講：訓練不了人工智慧？你可以訓練你自己 (上) — 神奇咒語與提供更多資訊](https://www.youtube.com/watch?v=A3Yx35KrSN0&list=PLJV_el3uVTsPz6CTopeRp2L2t4aL_KgiI&index=4)開始瞭解。相信我，你不用是電機專業也能看懂。
