## 宿舍網路規定 / Dormitory Network Rules

1. 每日流量限制：下載 7GB、上傳 4GB，超過流量會斷網。可至 top100.nchu.edu.tw 查詢是否超流。
   Daily traffic limit: 7 GB download, 4 GB upload. Exceeding the limit will result in network disconnection. Check your usage at top100.nchu.edu.tw.

2. 若被斷網，會於隔日早上 5:00 復網。
   If disconnected, the network will be restored at 5:00 AM the next day.

3. 被斷網時，可至 ipblock.nchu.edu.tw/cblocklist_public.php 查詢斷網原因。
   If disconnected, check the reason at ipblock.nchu.edu.tw/cblocklist_public.php.

4. 寧靜樓層於凌晨 1:00 斷網。
   Quiet floors: network is disabled at 1:00 AM.

5. 禁止使用 P2P 軟體（如：比特彗星、迅雷）。
   P2P software (e.g., BitComet, Thunder) is prohibited.

6. 新學年開宿或暑宿開始時需重新申請，申請成功後該學年或暑假期間不需重複申請。
   Re-apply at the start of each academic year or summer session. Once approved, no re-application is needed during that period.

## 申請前的準備 / Preparation Before Applying

### 1. 所需設備 / Required Equipment

學校宿舍僅提供網路孔（RJ45 乙太網路埠），請自行準備以下設備：

The dormitory provides only an Ethernet (RJ45) port. Please prepare the following:

- **網路線 (Ethernet cable)** — 一條足夠長度的 Cat5e 或 Cat6 網路線
- **有線網路轉接器 (USB Ethernet adapter)** — 如果您的電腦沒有內建 RJ45 網路孔（如輕薄筆電、MacBook Air 等），需購買 USB-A 或 USB-C 轉 RJ45 的轉接器
- **WiFi 機 (Wireless router / AP)** — 如果您需要無線上網，可自行準備並設定 WiFi 分享器

[示意圖說明：此處顯示一條藍色網路線、一個 USB-A 轉 RJ45 轉接器、一個 USB-C 轉 RJ45 轉接器，以及一台小型 WiFi 路由器]

### 2. 選擇連線方式 / Choose Connection Type

#### 有線網路 Wired Connection
- 關閉電腦的 WiFi
- 將網路線一端插入牆上網路孔，另一端插入電腦（或透過轉接器連接）
- 電腦沒有網路孔者請購買有線網路轉接器

#### 使用 WiFi 機 Wireless Connection
- 自備 WiFi 路由器
- 找各棟宿舍網管協助設定與處理

[示意圖說明：此處分左右兩側對比——左側為筆電直接插網路線的有線方案，右側為筆電連接 WiFi 路由器的無線方案]

### 3. 設定為「自動取得 IP」/ Set IP to DHCP

#### Windows 設定
1. 開啟「設定」→「網路和網際網路」
2. 點選「乙太網路」
3. 點選目前使用的乙太網路連線
4. 找到「IP 設定」，確認設為「自動 (DHCP)」
5. 若不是，請點選「編輯」更改為自動取得 IP

[示意圖說明：此處為 Windows 設定畫面的截圖，依序標示 1→2→3→4 的步驟指引，最終畫面顯示 IP 設定為自動(DHCP) 的選項]

#### MacOS 設定
1. 開啟「系統偏好設定」→「網路」
2. 選擇「USB 10/100 LAN」或「乙太網路」
3. 確認「設定 IPv4」為「使用 DHCP」
4. 若不是，請更改為 DHCP

## 申請流程 / Application Procedure

### Step 1: 開啟申請網頁

開啟瀏覽器（Chrome、Safari 等），在網址列輸入：

```
140.120.4.13
```

[示意圖說明：此處為瀏覽器截圖，網址列中已輸入 140.120.4.13，頁面載入中興大學宿舍網路申請系統]

### Step 2: 登入系統

輸入您的學校信箱與密碼，確認無誤後點選「送出」。

[示意圖說明：此處為登入表單截圖，包含「信箱」輸入欄位、「密碼」輸入欄位，以及「送出」按鈕]

### Step 3: 填寫電話號碼

輸入您的聯絡電話號碼，完成後點選「儲存」。

[示意圖說明：此處為電話號碼輸入頁面截圖，包含一個電話號碼輸入框和「儲存」按鈕]

### Step 4: 確認申請成功

頁面顯示申請成功訊息即代表已完成申請。

[示意圖說明：此處為申請成功頁面截圖，顯示成功提交的確認訊息]

### Step 5: 啟用網路

有線網路使用者請重新開機電腦，並重新插拔網路線。等待一段時間後即可上網。

> ※ 每次電腦重開機或重新插拔網路線，都需要稍等片刻才會取得 IP 並連線。

## 疑難排解 / Troubleshooting

若按照上述步驟操作後仍無法連線：

1. 至各棟宿舍網管室外填寫問題單，網管將於值勤時間前往處理。
2. 於網管值勤時間直接到網管室尋求協助。

> ※ 剛開學時申請人數眾多，問題不一定能當週處理完畢，敬請見諒。

## 附錄：查詢網卡卡號（MAC 位址）/ Appendix: Find MAC Address

### Windows
1. 開啟「設定」→「網路和網際網路」
2. 點選「乙太網路」
3. 點選目前使用的乙太網路連線
4. 向下捲動至最底部，找到「實體位址 (MAC)」

[示意圖說明：此處為 Windows 乙太網路設定頁面的截圖，畫面底部顯示「實體位址 (MAC)」欄位及其對應數值]

### MacOS
1. 開啟「系統偏好設定」→「網路」
2. 選擇「USB 10/100 LAN」或「乙太網路」
3. 點選「進階」→「硬體」
4. 即可查看「MAC 位址」

**快捷方式**：按住 Option（Alt）鍵，同時點擊螢幕右上角的網路圖示，即可直接顯示網卡卡號。

[示意圖說明：此處為 MacOS 網路偏好設定截圖，依序標示進入「進階」→「硬體」頁面後顯示 MAC 位址的位置；另一張示意圖顯示按住 Option 鍵點選選單列網路圖示後彈出的資訊畫面，其中包含 MAC 位址]
