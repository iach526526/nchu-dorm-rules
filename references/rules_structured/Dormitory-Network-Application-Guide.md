# National Chung Hsing University Dormitory Network Application Guide

## Section 1: Dormitory Network Rules

### Keywords

traffic limit, disconnection, P2P, quiet floor

### Chinese Text

1. 每日流量限制：下載 7GB、上傳 4GB，超過流量會斷網。可至 top100.nchu.edu.tw 查詢是否超流。
2. 若被斷網，會於隔日早上 5:00 復網。
3. 被斷網時，可至 ipblock.nchu.edu.tw/cblocklist_public.php 查詢斷網原因。
4. 寧靜樓層於凌晨 1:00 斷網。
5. 禁止使用 P2P 軟體（如：比特彗星、迅雷）。
6. 新學年開宿或暑宿開始時需重新申請，申請成功後該學年或暑假期間不需重複申請。

### English Translation

1. Daily traffic limit: 7 GB download, 4 GB upload. Exceeding the limit will result in network disconnection. Check your usage at top100.nchu.edu.tw.
2. If disconnected, the network will be restored at 5:00 AM the next day.
3. If disconnected, check the reason at ipblock.nchu.edu.tw/cblocklist_public.php.
4. Quiet floors: network is disabled at 1:00 AM.
5. P2P software (e.g., BitComet, Thunder) is prohibited.
6. Re-apply at the start of each academic year or summer session. Once approved, no re-application is needed during that period.

### Structured Summary

- Daily traffic: 7GB down, 4GB up; exceeding = disconnection
- Restored next day 5:00 AM
- Check block reason at ipblock.nchu.edu.tw
- Quiet floors: network off at 1:00 AM
- No P2P software
- Re-apply each academic year/summer session

### Penalty Points

None specified

## Section 2: Preparation Before Applying

### Keywords

equipment, Ethernet cable, adapter, router, DHCP

### Chinese Text

所需設備：網路線（Cat5e 或 Cat6）、有線網路轉接器（USB Ethernet adapter）、WiFi 機（Wireless router / AP）。
選擇連線方式：有線網路（關閉 WiFi，網路線插入電腦）或使用 WiFi 機。
設定為「自動取得 IP」：Windows 設定 DHCP；MacOS 設定 DHCP。

### English Translation

Required equipment: Ethernet cable (Cat5e or Cat6), USB Ethernet adapter (if needed), Wireless router/AP.
Choose connection type: Wired (disable WiFi, plug in Ethernet) or Wireless (router).
Set IP to DHCP: Windows/MacOS.

### Structured Summary

- Equipment needed: Ethernet cable (Cat5e/Cat6), USB Ethernet adapter (for laptops without RJ45), wireless router
- Wired: disable WiFi, plug in Ethernet
- Wireless: use own router; see network admin for setup
- Set IP to DHCP

### Penalty Points

None specified

## Section 3: Application Procedure

### Keywords

registration, 140.120.4.13, login, student email

### Chinese Text

Step 1: 開啟瀏覽器，輸入 140.120.4.13
Step 2: 輸入學校信箱與密碼登入
Step 3: 填寫電話號碼
Step 4: 確認申請成功
Step 5: 啟用網路（重新開機並重新插拔網路線）

### English Translation

Step 1: Open browser, enter 140.120.4.13
Step 2: Login with school email and password
Step 3: Enter phone number
Step 4: Confirm success
Step 5: Restart computer and reconnect Ethernet cable

### Structured Summary

- Open browser to 140.120.4.13
- Login with school email
- Enter phone number
- Confirm application
- Restart computer and reconnect cable to activate

### Penalty Points

None specified

## Section 5: Appendix: Find MAC Address

### Keywords

MAC address, Windows, MacOS

### Chinese Text

Windows: 設定 → 網路和網際網路 → 乙太網路 → 實體位址 (MAC)
MacOS: 系統偏好設定 → 網路 → 進階 → 硬體 → MAC 位址

### English Translation

Windows: Settings → Network & Internet → Ethernet → Physical address (MAC)
MacOS: System Preferences → Network → Advanced → Hardware → MAC Address

### Structured Summary

- Find MAC address for network registration
